
# "Neues Video starten" -- Reset-Button fuer den Universal Video Creator

## Was wird gemacht

Ein gut sichtbarer Reset-Button wird zur Wizard-Oberflaeche hinzugefuegt, der alle gespeicherten Antworten und den Fortschritt komplett zuruecksetzt und den Nutzer zurueck zum Anfang bringt.

## Wo erscheint der Button

- Neben dem bestehenden "Zurueck"-Button (oben links im Wizard)
- Wird ab Schritt 1 angezeigt (also sobald man eine Kategorie gewaehlt hat)
- Auch waehrend/nach dem Rendering sichtbar, damit man von der 92%-Ansicht zurueck zum Start kann

## Was der Button macht

1. Loescht `universal-video-wizard-state` aus localStorage
2. Loescht `universal-video-consultant-state` aus localStorage
3. Setzt alle State-Variablen zurueck (Kategorie, Modus, Schritt, Beratungsergebnis, Fehler)
4. Nutzt die bereits vorhandene `handleBackToCategory`-Logik

## Technische Details

### Datei: `src/components/universal-video-creator/UniversalVideoWizard.tsx`

**Aenderung 1:** Die bestehende `handleBackToCategory`-Funktion wird erweitert, um auch `isAutoGenerating` und `autoGeneratedProject` zurueckzusetzen:

```typescript
const handleResetWizard = () => {
  setSelectedCategory(null);
  setGenerationMode(null);
  setCurrentStep(0);
  setError(null);
  setIsAutoGenerating(false);
  setAutoGeneratedProject(null);
  setConsultationResult(null);
  localStorage.removeItem('universal-video-wizard-state');
  localStorage.removeItem('universal-video-consultant-state');
};
```

**Aenderung 2:** Ein neuer Button wird neben dem "Zurueck"-Button eingefuegt (ca. Zeile 309-319), sichtbar auf allen Schritten > 0:

```typescript
{currentStep > 0 && (
  <div className="mb-4 flex items-center gap-2">
    {currentStepId !== 'generating' && (
      <Button variant="ghost" size="sm" onClick={handleBack}>
        <ArrowLeft /> Zurueck
      </Button>
    )}
    <Button variant="outline" size="sm" onClick={handleResetWizard}>
      <RotateCcw /> Neues Video starten
    </Button>
  </div>
)}
```

- Nur eine Datei wird bearbeitet
- Kein Backend-/Datenbank-Aenderung noetig
- Der Button ist auch waehrend des Renderings verfuegbar, damit man aus dem 92%-Zustand herauskommt
